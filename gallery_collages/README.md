# Gallery collages — Local to Global phase

Put one collage image per school here (e.g. `GSSS-DUMAR.jpg`).

The filename must match the "Collage image filename" entered in `admin.html`
for that school. The Gallery section on the website picks it up automatically.

(The Punjab 2025–26 collages live in `gallery_school_collage/` — don't mix them.)
